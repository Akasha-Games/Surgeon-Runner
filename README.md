
# 🎮 Surgeon Runner

A Unity-based endless runner game developed in C# as part of my game development journey. This project focuses on learning gameplay programming, game systems architecture, quality assurance, object-oriented programming, Unity workflows, and player experience design.

---

## 📌 Project Overview

Surgeon Runner is an endless runner game where players navigate through a hospital-themed environment, avoid obstacles, collect syringes, and achieve the highest score possible.

The project serves as a practical learning platform for implementing core game development concepts including player movement, collectibles, audio systems, collision detection, animation, and gameplay mechanics.
